# Using Open OnDemand

Open OnDemand provides the following features.

* File transfer and editing
* Web terminal access
* Running and managing interactive applications, such as remote desktops
* Job submission and management

Open OnDemand supports major web browsers such as Google Chrome, Mozilla Firefox, and Microsoft Edge (note: Internet Explorer 11 is not supported). Chrome natively supports copying and pasting text in remote desktop sessions and similar environments. For this reason, Chrome is recommended.
